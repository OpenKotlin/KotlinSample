## 结构

项目按照功能分包

* MainActivity ： 项目入口
    
    * getStarted kotlin 基本功能
    
    * operations kotlin 操作符
    
    * extension kotlin 扩展特性
    
    * recyclerView kotlin 版 RecyclerView 例子
